01 — Visão Geral
Passkey ≠ link por e-mail. Após registrar uma passkey, o usuário pode entrar no painel sem digitar senha, usando biometria/PIN do dispositivo. Nenhum e-mail de acesso é enviado no login.

O que é Laravel Passkeys

Pacote first-party da equipe Laravel (passkeys-server) para WebAuthn: Touch ID, Face ID, Windows Hello, YubiKey.

02 — Ganhos Possíveis

Experiência do usuário

  • Login passwordless — biometria em 1–2 segundos
  • Múltiplos dispositivos — MacBook, iPhone, YubiKey
  • Autofill nativo — Chrome/Safari sugerem passkey
  • Self-service — revogar passkey no perfil Filament

Segurança

  • Anti-phishing — credencial amarrada ao domínio
  • Sem senha reutilizada — reduz credential stuffing
  • authorizeLoginUsing — paridade com Login.php customizado
  • Eventos — PasskeyRegistered, PasskeyVerified, PasskeyDeleted
  • Reauth bound — confirmar ações críticas sem redigitar senha

O que NÃO muda automaticamente

  • Fluxo de aprovação manual de novos usuários
  • RBAC FilaTeams + Spatie Permission
  • Filament Security e tokens Sanctum
  • MFA TOTP — até decisão explícita de substituir/combinar
