# Comandos Úteis
 
- `vendor/bin/sail artisan make:filament-resource --view`: Cria recurso Filament com view page.
- `vendor/bin/sail artisan config:clear`: Limpa cache de configuração (essencial após remover pacotes).
- `vendor/bin/sail composer install`: Sincroniza dependências.
- `vendor/bin/sail artisan migrate`: Executa migrations.
